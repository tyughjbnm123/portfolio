MARTECH CANVAS // 節點編輯沙盒

// 視覺化 Martech 節點編輯器

本沙盒展示了我為企業建置的 Hosset 虎勢 CRM LINE OA 自動化工作流。 在下方畫布中，您可以拖曳節點、點擊節點查看配置，或點擊頂部「運行工作流」，在右側控制台觀看即時自動化運行日誌。

HOSSET_LINE_CRM_WORKFLOW.json
1. LINE 掃碼觸發
線下活動/DM 掃描 QR Code，自動預填「我要綁定會員」關鍵字。
2. Omnichat 機器人
辨識關鍵字，推送 Rich Message 綁定引導圖卡。
3. Cyberbiz 串接
呼叫 Cyberbiz API，驗證手機號碼並綁定 OpenID 會員卡。
4. 自動發券與標籤
自動派發 $100 綁定禮券，並於 Omnichat 上打上「已綁定會員」標籤。
5. 數據分析看板
即時將綁定率、發券率與核銷數據同步至 Looker Studio 儀表板。
